# LojaDados - SQL Server

Projeto de banco de dados relacional desenvolvido utilizando SQL Server.

## Objetivo

Criar uma estrutura de banco para gerenciamento de clientes, produtos e vendas.

## Tecnologias utilizadas

- SQL Server
- SQL Server Management Studio (SSMS)

## Estrutura do Banco

O projeto possui as tabelas:

- Cliente
- Produto
- Venda
- Item_Venda

## Conceitos aplicados

- Criação de banco de dados
- Modelagem relacional
- Chaves primárias e estrangeiras
- Relacionamentos entre tabelas
- INSERT, UPDATE e DELETE
- SELECT e filtros
- INNER JOIN
- GROUP BY
- Views
- Stored Procedures

## Relacionamentos

Cliente possui várias vendas.

Venda possui vários produtos através da tabela Item_Venda.
